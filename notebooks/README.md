# Notebooks

This directory contains Jupyter notebooks for the portfolio optimization analysis.

## Notebooks

### 01_data_preprocessing_eda.ipynb
**Task 1: Preprocess and Explore the Data**

This notebook covers:
- Extraction of historical financial data using YFinance
- Data cleaning and quality assessment
- Exploratory Data Analysis (EDA) with visualizations
- Stationarity testing using Augmented Dickey-Fuller (ADF) test
- Risk metrics calculation (Value at Risk, Sharpe Ratio)
- Identification of trends, patterns, and anomalies
